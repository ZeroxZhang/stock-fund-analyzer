# 技术面分析框架参考 (Technical Analysis Framework Reference)

> 本文档为《股票基金综合分析框架 V2.0》中**第四部分：技术分析**的深度补充与扩展规范，供智能体在进行技术面分析时作为专业方法论参考。实际分析中应根据具体标的（股票/ETF/期货等）选择适用的指标与周期，灵活组合运用。

---

## 一、分析框架总览

技术分析遵循「**先趋势、后结构、再指标**」的三层递进逻辑：

```
第一层：趋势确认（道氏理论）
  └─ 主要趋势（Primary）→ 次要趋势（Secondary）→ 短期趋势（Tertiary）

第二层：价格结构（波浪理论）
  └─ 推动浪计数 → 调整浪识别 → 多周期共振验证

第三层：指标信号（量价/动量/趋势/波动率）
  └─ 各指标一致性检验 → 信号权重裁定 → 最终结论
```

---

## 二、第一层：趋势分析（道氏理论扩展）

### 2.1 趋势定义与判定标准

| 趋势类型 | 时间周期 | 判定标准 | 确认信号 |
| ------ | ----- | ------ | ------ |
| 主要趋势（Primary） | 1年至数年 | 持续创新高/新低 | 成交量放大确认 |
| 次要趋势（Secondary） | 3周至数月 | 回调<前一波段的33%-66% | 反弹缩量、下跌放量 |
| 短期趋势（Tertiary） | 1天至3周 | 连续高低点序列 | 均线多头/空头排列 |

### 2.2 趋势线与通道

**上升趋势线画法：**
- 连接两个依次上升的回调低点
- 第三个低点触及趋势线则趋势确认
- 跌破上升趋势线超过3%视为趋势破坏

**下降趋势线画法：**
- 连接两个依次下降的反弹高点
- 第三个高点触及趋势线则趋势确认
- 升破下降趋势线超过3%视为趋势破坏

**通道线：**
- 与趋势线平行的平行线，连接第一个显著的高点/低点
- 通道突破通常意味着加速（顺趋势）或反转（逆趋势）

### 2.3 支撑与阻力位识别

| 类型 | 定义 | 验证条件 | 强度判断 |
| -- | -- | -- | -- |
| 支撑位 | 价格下跌时遇到买盘支撑的区域 | 至少两次触及不破 | 触及次数越多越强 |
| 阻力位 | 价格上涨时遇到卖盘压力的区域 | 至少两次触及不破 | 成交量越大越强 |
| 心理关口 | 整数关口（100/200/1000等） | 价格行为确认 | 短期有效 |
| 黄金分割 | 38.2%/50%/61.8%回撤位 | 价格行为确认 | 中期有效 |

**止损位设定原则：**
- 保守止损：主要支撑/阻力下方3-5%
- 正常止损：近期低点/高点下方2-3%
- 积极止损：趋势线/通道边界
- **风险回报比**必须 ≥ 1:2 才值得操作

## 三、第二层：波浪理论（艾略特波浪精解）

### 3.1 基本浪型规则

**推动浪（Impulse Waves）必须满足：**
1. 浪2的回撤不能吃掉浪1的100%
2. 浪3永远不是最短的一浪
3. 浪4不能与浪1重叠（除非在斜向三角中）
4. 交替原则：浪2与浪4的形态必须交替（简单/复杂）

**调整浪（Corrective Waves）三种基本形态：**

| 形态 | 特征 | 标记 | 出现位置 |
| -- | -- | -- | -- |
| 之字形（Zigzag） | 快速深幅回落，3-3-5结构 | A-B-C | 2浪、4浪、B浪 |
| 平坦型（Flat） | 横向整理，3-3-5结构 | A-B-C | 4浪、X浪、B浪 |
| 三角型（Triangle） | 收敛或扩散，3-3-3-3-3结构 | A-B-C-D-E | 4浪、B浪、终浪 |

### 3.2 波浪比率与目标位计算

**黄金分割在波浪中的应用：**

| 比率 | 应用场景 |
| -- | -- |
| 61.8%（0.618） | 推动浪中2/4浪回撤比例；调整浪中B浪回撤A浪 |
| 78.6%（0.786） | 强势调整回撤比例 |
| 38.2%（0.382） | 弱势调整回撤比例 |
| 127.2%（1.272） | 推动浪3的最小目标 |
| 161.8%（1.618） | 推动浪3的强势目标；C浪=1.618×A浪 |
| 261.8%（2.618） | 推动浪5的目标（弱势） |
| 423.6%（4.236） | 推动浪5的目标（强势） |

**等距测量法：**
- 当3浪终点难以用比率预测时，使用1浪起点到3浪终点的距离，从4浪终点投射同样的距离，作为5浪目标
- 公式：5浪目标 = 4浪终点 + (3浪终点 − 1浪起点)

### 3.3 多周期共振验证

| 周期 | 用途 | 共振条件 |
| -- | -- | -- |
| 周线 | 确定主要趋势方向 | 与日线同向 |
| 日线 | 确定次级趋势与入场点 | 与60分钟同向 |
| 60分钟 | 确定短期入场时机 | 与日线反向时提供更好的赔率 |

**共振类型：**
- **正向共振**：所有周期同向上涨 → 趋势强劲，高胜率顺势操作
- **逆向共振**：小周期与主趋势反向 → 提供更好的回调入场点位
- **背离共振**：不同周期计数矛盾 → 降低仓位或等待确认

### 3.4 波浪计数评估标准

| 置信度 | 条件 |
| -- | -- |
| 高 | 满足所有基本规则、多个比率目标汇合、价格行为完全吻合 |
| 中 | 满足所有基本规则、少数比率目标汇合 |
| 低 | 勉强满足基本规则、替代计数同样有效 |

**替代计数（Alternative Count）：**
- 始终保留一个替代数法
- 当主计数失效条件触发时，自动切换至替代计数
- 例：若3浪不能短于1浪，则主计数失效，切换至延伸3浪计数

## 三（续）、K线形态分析（补充）

### 3.5 反转形态

| 形态 | 周期 | 确认条件 | 信号强度 |
| -- | -- | -- | -- |
| 锤头线（Hammer） | 短期 | 出现在下跌后，下影线≥实体2倍，无上影线 | 中 |
| 射击星（Shooting Star） | 短期 | 出现在上涨后，上影线≥实体2倍，无下影线 | 中 |
| 吞没形态（Engulfing） | 短期 | 后一根K线完全包裹前一根，伴随放量 | 强 |
| 乌云盖顶（Dark Cloud Cover） | 短期 | 出现在上涨后，第二根K线高开低收至前实体50%以下 | 强 |
| 锤子线/上吊线（相同形态，不同位置） | 短期 | 实体小、下影线长，出现在顶部时为上吊线（看跌） | 中 |
| 早晨之星（Morning Star） | 次级 | 三K线组合，下跌后出现十字/小实体，再出现中阳线吞噬 | 强 |
| 黄昏之星（Evening Star） | 次级 | 三K线组合，上涨后出现十字/小实体，再出现中阴线吞噬 | 强 |

### 3.6 持续形态

| 形态 | 周期 | 价格目标计算 | 信号强度 |
| -- | -- | -- | -- |
| 旗形（Flag） | 短期 | 旗杆长度×突破方向 | 中 |
| 三角旗形（Pennant） | 短期 | 旗杆长度×突破方向 | 中 |
| 对称三角形 | 次级 | 突破方向为原有趋势，幅度≥三角形最宽度 | 中 |
| 上升三角形 | 次级 | 突破后幅度≥三角形最宽度，向上概率大 | 中强 |
| 下降三角形 | 次级 | 突破后幅度≥三角形最宽度，向下概率大 | 中强 |
| 矩形整理 | 次级 | 区间宽度为目标位 | 中 |

## 四、第三层：技术指标系统

### 4.1 趋势类指标

#### 4.1.1 移动平均线（MA）

**参数配置参考：**

| 周期组合 | 适用场景 | 用途 |
| -- | -- | -- |
| 5/10/20 | 短线交易 | 快速趋势判断、入场信号 |
| 20/60/120 | 中线交易 | 趋势过滤、动态支撑/阻力 |
| 50/100/200 | 长线交易 | 主要趋势确认 |
| EMA(12,26) | MACD基础 | 趋势与动量结合 |
| EMA(9) | MACD信号线 | 交叉信号触发 |

**均线用法：**
- **多头排列**：短期 > 中期 > 长期，向上发散 → 上升趋势
- **空头排列**：短期 < 中期 < 长期，向下发散 → 下降趋势
- **黄金交叉**：短期均线上穿长期均线 → 买入信号（需成交量确认）
- **死亡交叉**：短期均线下穿长期均线 → 卖出信号（需成交量确认）
- **均线粘合**：多条均线收敛 → 酝酿突破，粘合越久爆发越大

**均线的支撑与阻力：**
- 上升趋势中，价格回踩20日/60日均线是买入机会
- 重要均线被有效突破（超过3%）视为趋势转变信号

#### 4.1.2 MACD（指数平滑异同移动平均线）

**参数：** 12, 26, 9（标准参数，可根据标的特性调整）

**核心用法：**

| 信号 | 看涨条件 | 看跌条件 |
| -- | -- | -- |
| **MACD交叉** | DIF线从下上穿DEA线 | DIF线从上死叉DEA线 |
| **零轴交叉** | DIF从负转正并上穿零轴 | DIF从正转负并下穿零轴 |
| **背离** | 价格创新低，MACD未创新低 | 价格创新高，MACD未创新高 |
| **柱状图收缩** | 红柱缩短预示下跌动能减弱 | 绿柱缩短预示上涨动能减弱 |

**背离判断标准：**
- 顶背离：价格创新高，DIF未创新高（≥2次）
- 底背离：价格创新低，DIF未创新低（≥2次）
- 背离须结合其他信号确认，单独背离不可作为入场唯一依据

#### 4.1.3布林带（Bollinger Bands）

**参数：** 20日均线 ± 2倍标准差

| 带宽特征 | 市场状态 | 含义 |
| -- | -- | -- |
| 带宽收窄（<50%历史均值） | 低波动 | 酝酿突破，窄幅整理后必有大幅波动 |
| 带宽扩张 | 高波动 | 趋势加速或趋势反转 |
| 带宽正常 | 常态 | 趋势延续 |

**价格触及布林带：**
- 触及上轨：强势，但需警惕回调
- 触及下轨：弱势，但需警惕反弹
- 强势市场中，价格沿上轨行走，可持有
- 弱势市场中，价格沿下轨行走，应观望

**布林带结合均线：**
- 布林中轨 = 20日均线 = 动态趋势线
- 价格回撤至中轨企稳 → 买入机会（上升趋势中）
- 反弹至中轨受阻 → 卖出机会（下降趋势中）

### 4.2 动量类指标

#### 4.2.1 RSI（相对强弱指数）

**参数：** 14日（标准），可调整为6/12/26

**超买超卖判断：**

| RSI区间 | 市场状态 | 可能含义 |
| -- | -- | -- |
| RSI > 80 | 极度超买 | 警惕顶部，可能进入调整 |
| RSI > 70 | 超买 | 上涨动能边际减弱 |
| RSI 40-60 | 中性区间 | 无明确方向 |
| RSI < 30 | 超卖 | 下跌动能边际减弱 |
| RSI < 20 | 极度超卖 | 警惕底部，可能出现反弹 |

**RSI背离（与MACD背离判断标准相同）：**
- 价格创新低，RSI未创新低 → 底背离（买入机会）
- 价格创新高，RSI未创新高 → 顶背离（卖出机会）

**RSI趋势线：**
- 连接RSI的两个以上高点/低点形成趋势线
- RSI突破自身趋势线先于价格突破 → 先行信号

**RSI的局限性：**
- 在强势趋势中，RSI可能长时间维持在超买/超卖区域
- 趋势行情中，应跟随趋势而非逆势抄顶/抄底

#### 4.2.2 KDJ（随机指标）

**参数：** 9, 3, 3

**核心用法：**

| 信号 | 看涨 | 看跌 |
| -- | -- | -- |
| **金叉** | K线从下上穿D线，且位于20以下 | K线从下上穿D线，且位于80以上（弱化） |
| **死叉** | K线从上死叉D线，且位于80以上 | K线从上死叉D线，且位于20以下（弱化） |
| **背离** | 与MACD/RSI背离判断一致 | 同左 |

**KDJ超买超卖：**
- KDJ > 80：超买区域，高频交易者可在此区域关注死叉
- KDJ < 20：超卖区域，高频交易者可在此区域关注金叉
- **注意**：在强势趋势中，KDJ可能反复在超买区域金叉死叉，此时应主要跟随趋势方向

#### 4.2.3 CCI（顺势指标）

**参数：** 14日或20日

**用法：**

| CCI区间 | 含义 |
| -- | -- |
| CCI > +100 | 强势，可能延续上涨（顺趋势操作） |
| CCI < -100 | 弱势，可能延续下跌（顺趋势操作） |
| CCI 从下上穿 +100 | 新的上升趋势确立 |
| CCI 从上下穿 -100 | 新的下降趋势确立 |
| CCI 背离 | 与RSI类似，但更灵敏 |

**CCI特别适合：**
- 趋势行情中的回调买入/反弹卖出
- 商品期货、货币对等高波动标的

### 4.3 量价类指标

#### 4.3.1 成交量分析原则

**量价关系四大法则：**

| 法则 | 描述 | 信号 |
| -- | -- | -- |
| **量增价涨** | 成交量放大，价格上涨 | 确认趋势，稳健信号 |
| **量增价跌** | 成交量放大，价格下跌 | 确认下跌趋势 |
| **量缩价涨** | 成交量萎缩，价格上涨 | 趋势可能逆转（顶部预警） |
| **量缩价跌** | 成交量萎缩，价格下跌 | 底部区域，可能反转 |

**支撑/阻力的成交量确认：**
- 向上突破阻力位时放量（量能≥过去10日均量的1.5倍）→ 有效突破
- 向上突破阻力位时缩量 → 假突破预警
- 向下突破支撑位时放量 → 有效跌破
- 向下突破支撑位时缩量 → 可能是洗盘

**OBV（能量潮）：**
- 计算：OBV = 前期OBV + 今日成交量（上涨时）或 -今日成交量（下跌时）
- 用途：辅助确认趋势，OBV创新高 → 上涨动能充足；OBV创新低 → 下跌动能充足
- OBV与价格背离 → 预警趋势可能反转

#### 4.3.2 筹码分布（Volume Profile）

**概念：** 不同价格区间的成交量分布

**关键价位识别：**
- **高量柱**：该价格区间成交最密集，形成密集筹码区
- **低量柱**：该价格区间成交稀疏，是潜在支撑/阻力的真空区
- **筹码峰**：密集成交区形成山峰，对价格有吸引力或排斥力

**缺口理论：**
| 缺口类型 | 特征 | 操作意义 |
| -- | -- | -- |
| 普通缺口 | 成交量低，很快被回补 | 无分析意义 |
| 突破缺口 | 成交量放大当日出现，不被回补 | 确认趋势展开 |
| 持续缺口 | 出现在趋势中途，不被回补 | 持续趋势，明确目标位 |
| 衰竭缺口 | 趋势末端出现，成交量异常放大 | 趋势即将反转 |

### 4.4 波动率指标

#### 4.4.1 ATR（平均真实波幅）

**用途：**
- **止损设置**：建议止损位 = 入场价 - 1.5×ATR（做多）或 + 1.5×ATR（做空）
- **仓位计算**：根据ATR计算每笔交易愿意承受的亏损金额来反推仓位
- **波动判断**：ATR数值越大，市场波动越剧烈

**ATR与趋势判断：**
- 上升趋势中，ATR通常上升（波动加大）
- 盘整期间，ATR通常收缩（波动降低）
- **窄幅整理+ATR收缩** → 酝酿突破

#### 4.4.2 历史波动率（HV）与隐含波动率（IV）

| 指标 | 定义 | 用途 |
| -- | -- | -- |
| 历史波动率（HV） | 过去N日收益率的标准差年化 | 评估过去实际波动水平 |
| 隐含波动率（IV） | 期权价格反推的波动率预期 | 市场对未来波动的预期 |

**波动率交易逻辑：**
- IV低 → 期权便宜 → 适合买方
- IV高 → 期权昂贵 → 适合卖方
- VIX（恐慌指数）> 30 → 市场恐慌，可能出现超卖反弹机会
- VIX < 15 → 市场平静，趋势可能延续

## 五、综合信号裁定规则

### 5.1 多指标一致性裁定

当多个指标给出方向性信号时，按以下权重裁定：

| 优先级 | 指标类型 | 代表指标 | 权重说明 |
| -- | -- | -- | -- |
| 1 | 趋势（核心） | MA多头排列、趋势线突破 | 最优先，价格包容一切 |
| 2 | 结构（波浪） | 波浪计数、形态突破 | 确认趋势发展的阶段 |
| 3 | 动量 | MACD/RSI交叉、背离 | 确认动能方向 |
| 4 | 量价 | 成交量确认、OBV | 验证价格真实性 |
| 5 | 波动率 | ATR收缩/扩张 | 判断突破有效性 |

### 5.2 指标矛盾时的处理

| 矛盾场景 | 处理原则 |
| -- | -- |
| 趋势指标看多，动量指标看空 | 以趋势为主，等待动量修复（底背离可能提供更好入场价） |
| 趋势指标看空，动量指标看多 | 以趋势为主，等待动量背离确认顶部 |
| 短周期与长周期矛盾 | 大周期趋势优先，小周期提供入场点位 |
| K线形态与技术指标矛盾 | 短期服从技术指标，中期关注形态 |

### 5.3 交易信号强度分级

| 等级 | 信号条件 | 操作建议 |
| -- | -- | -- |
| **A（强确认）** | 趋势+波浪+动量+量价全部同向 | 重仓顺势操作 |
| **B（确认）** | 三项以上同向 | 顺势轻仓或标准仓 |
| **C（中性）** | 两项同向，两项矛盾 | 观望或轻仓试单 |
| **D（矛盾）** | 各项指标方向混乱 | 空仓等待明确信号 |
