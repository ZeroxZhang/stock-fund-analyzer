# Stock & Fund Analyzer Skill

A comprehensive stock and fund analysis skill compatible with Claude/Trae, combining Value Investing (Graham, Buffett, Howard Marks) and Technical Analysis (Dow Theory, Elliott Wave, CAN SLIM) frameworks.

## Features

When analyzing a stock or fund, this skill will:

1. Fetch real-time financial, market, institutional expectations, fund flow, and macro/industry data
2. Conduct in-depth macro and industry environment assessment
3. Perform value investing analysis (Graham Number, DCF, Moat Analysis, Cyclical Assessment)
4. Execute technical analysis (Dow Theory, Elliott Wave, CAN SLIM Model)
5. Calculate weighted fair value price ranges
6. Generate investment memorandum with clear conclusions (Buy/Hold/Sell)

## Data Collection Scripts (`scripts/`)

The `scripts/__init__.py` provides complete Python reference implementations:

| Function | Purpose | Data Source |
|----------|---------|-------------|
| `fetch_stock_info()` | Stock basics, valuation, analyst expectations | yfinance |
| `fetch_stock_financials()` | Revenue, profit, cash flow, ROE/ROIC | yfinance |
| `fetch_price_history()` | Historical OHLCV for technical analysis | yfinance |
| `fetch_etf_info()` | ETF basics, expense ratio, returns | yfinance |
| `fetch_etf_holdings()` | Top 10 ETF holdings | yfinance |
| `fetch_cn_stock_info()` | A-share stock information | akshare |
| `fetch_cn_money_flow()` | A-share fund flow data | akshare |
| `fetch_macro_indicators()` | Macro indicators (rates, FX, etc.) | yfinance/public APIs |
| `fetch_industry_index()` | SW industry indices | akshare |
| `calculate_technical_indicators()` | MA, RSI, MACD, Bollinger, ATR | numpy/pandas |

### Quick Start

```python
from scripts import fetch_stock_info, fetch_price_history, calculate_technical_indicators

# Get stock info
info = fetch_stock_info("AAPL")
print(f"Current Price: ${info['current_price']}")
print(f"P/E Ratio: {info['pe_ttm']}")

# Get price history with technical indicators
history = fetch_price_history("AAPL", period="1y")
indicators = calculate_technical_indicators(history)
print(indicators[['Close', 'MA20', 'RSI14', 'MACD']].tail())
```

## Technical Analysis Framework (`docs/`)

The `docs/technical-analysis-framework.md` provides comprehensive methodology:

- Dow Theory three-tier trend analysis
- Elliott Wave theory (impulse/corrective wave rules, ratios, counting)
- Candlestick patterns (reversal/continuation)
- Four categories of technical indicators (trend/momentum/volume/volatility)
- Multi-indicator signal reconciliation rules
- Technical analysis report template

## Usage

In Trae IDE or compatible agent environments, place `SKILL.md` in your project's skill directory (e.g., `.trae/skills/stock-fund-analyzer/`). Trigger with natural language:

- *"帮我分析一下特斯拉 (TSLA) 的投资价值"*
- *"看看 QQQ 现在的走势和估值情况"*
- *"使用综合分析框架评估一下英伟达"*
- *"Analyze Apple's investment potential"*
