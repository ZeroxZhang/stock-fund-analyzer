# 股票基金综合分析框架 (Stock & Fund Analyzer)

本技能用于对任意一只股票或基金进行**系统性、全方位、多维度**的分析，并输出具有明确投资结论的结构化研判报告。

## 触发条件 (When to invoke)
- 用户要求分析某只股票（如："帮我分析一下苹果 AAPL"）。
- 用户要求评估某只基金或 ETF（如："看看 QQQ 现在的投资价值"）。
- 用户询问某资产是否值得买入、卖出或持有。
- 用户提到"使用股票综合分析框架"或类似表述。

## 执行流程 (Execution Steps)

当你被调用来分析股票或基金时，你需要**严格**按照以下结构进行数据收集和分析，最终生成一份完整的 Markdown 报告。

### 1. 数据采集 (Data Collection)

你需要通过你的搜索工具（Search/WebSearch 等）或者金融数据工具实时获取以下最新数据。**若某项数据缺失，请明确标注"数据缺失"，切勿编造。所有数据需注明来源和时间。**

#### 1.1 数据获取方式

| 数据类型 | 推荐获取方式 | 说明 |
| -- | -- | -- |
| 美股基础数据、财务、估值、机构预期 | 使用 `yfinance` Python 库（免费） | 参考 `scripts/__init__.py` 中的 `fetch_stock_info()`、`fetch_stock_financials()` |
| A股基础数据、财务、估值 | 使用 `akshare` Python 库（免费） | 参考 `scripts/__init__.py` |
| 历史价格/K线（技术分析用） | 使用 `yfinance` 的 `fetch_price_history()` | 支持日/周/月/分钟级别 |
| 资金流向（北向/主力/融资融券） | 使用 `akshare` 的 `stock_individual_fund_flow()` | A股专用 |
| 宏观/行业数据 | akshare 专业接口 + WebSearch 补充 | 利率/CPI 建议通过专业终端获取 |
| 期权/衍生品数据 | WebSearch 搜索 "期权-put/call ratio 股票代码" | IV 等指标数据较难免费获取 |

> **参考脚本位置：** 本技能仓库中 `scripts/__init__.py` 提供了完整的数据采集参考实现，涵盖股票/ETF基础信息、财务数据、历史价格、资金流向、宏观行业等数据获取函数。智能体可参考该脚本的方法名称与返回结构，调用同等的 Python 工具执行数据获取。

#### 1.2 需采集的数据项

- **基础财务数据**：营收、净利润、毛利率、营业利润率、自由现金流(FCF)、资产负债率、EPS、**ROE**、**ROIC** 及相关增速趋势。
- **对比公司数据**：至少 3 家同行业可比公司的数据对比。
- **实时市场数据**：当前股价、市值、PE(TTM及前瞻)、PS(TTM)、PB、PEG、52周高低点、YTD及近1/3/6个月涨跌幅。
- **机构预期**：华尔街共识评级、目标价区间与中位数、EPS一致预期、评级调整历史。
- **资金流向**：主力资金近5日净流入、融资融券余额、筹码集中度（如适用）。
- **衍生品（如适用）**：IV、Put/Call Ratio。
- **宏观与行业**：利率环境、信用利差、美元指数、行业指数涨跌、行业PE分位。

### 2. 宏观与行业分析 (Macro & Industry Analysis)
评估当前宏观环境（货币政策、信贷、汇率、风险偏好），并对该资产所在行业进行五力分析及生命周期/景气度定位，列出核心的 2-3 个驱动因素或风险点。

### 3. 价值投资分析 (Value Investing Analysis)
- **格雷厄姆安全边际估值**：计算格雷厄姆数 `√(22.5 × 每股净资产 × 每股盈利)`，DCF内在价值估算（保守/中性/乐观场景），PEG估值。得出买入/持有/卖出的综合判断。
- **巴菲特护城河分析**：评估五项要素（品牌、转换成本、网络效应、成本优势、有效壁垒），并用财务数据验证（ROE/ROIC/利润率稳定性），评估护城河持续性风险，最终给出"宽/窄/无"护城河评级。
- **霍华德·马克斯周期思维**：判断经济/行业/信贷/库存/情绪周期位置，结合历史估值分位进行风险评估。
- **价值投资综合判断**：给出 A/B/C/D 明确的综合价值评级。

### 4. 技术分析 (Technical Analysis)

> **深度参考：** 本技能仓库中 `docs/technical-analysis-framework.md` 提供了一份完整的专业版技术面分析框架文档，涵盖道氏理论、波浪理论、K线形态、趋势/动量/量价/波动率指标体系等。当进行技术分析时，请同时参阅该文档作为方法论依据。

技术分析遵循「**先趋势、后结构、再指标**」的三层递进逻辑：

#### 4.1 道氏理论趋势分析
- **三层趋势判断**：主要趋势（Primary/周线）、次要趋势（Secondary/日线）、短期趋势（Tertiary/60分钟），每个周期给出方向、强度与确认依据。
- **趋势线与通道**：画法规范，突破3%以上视为破坏信号。
- **关键价位识别**：支撑位、阻力位（至少2次触及验证）、心理关口、黄金分割回撤位，附风险回报比。
- **止损位设定原则**：保守止损=主要支撑下方3-5%，正常止损=近期高低点下方2-3%。

#### 4.2 艾略特波浪分析
- **推动浪规则**：浪2不回撤100%于浪1、浪3非最短、浪4不与浪1重叠、交替原则。
- **调整浪三种形态**：之字形(Zigzag)、平坦型(Flat)、三角型(Triangle)，识别A-B-C结构。
- **波浪比率**：61.8%/78.6%/38.2%回撤位；127.2%/161.8%推动浪3目标；C浪=1.618×A浪。
- **多周期共振验证**：周线/日线/60分钟三个周期同向为强共振，逆向为提供更佳入场点位。
- **置信度评估**：高/中/低；替代计数；失效条件（跌破X则计数失效）。

#### 4.3 K线形态分析
- **反转形态**：锤头线、射击星、吞没、乌云盖顶、早晨之星、黄昏之星等，标注周期与信号强度。
- **持续形态**：旗形、三角旗形、对称三角形、上升/下降三角形、矩形整理等，给出价格目标计算方法。

#### 4.4 欧奈尔 CAN SLIM 七维度分析
按 C/A/N/S/L/I/M 七个维度进行逐项评估（权重参考：I 20%、C 15%、A 15%、L 15%、N 10%、S 10%、M 15%），得出总分及等级（A+ 到 D）。

#### 4.5 技术指标系统（四类指标综合运用）

| 指标类别 | 代表指标 | 核心用途 |
| -- | -- | -- |
| **趋势类** | MA(均线多头/空头排列)、MACD(零轴交叉)、布林带(支撑阻力) | 确定方向，顺势操作 |
| **动量类** | RSI(超买超卖/背离)、KDJ、CCI | 确认动能，发现转折 |
| **量价类** | 成交量法则、OBV、筹码分布、缺口理论 | 验证价格真实性 |
| **波动率类** | ATR(止损设置)、ATR收缩预警突破 | 判断波动，设定仓位 |

**指标综合裁定优先级：** 趋势(核心) > 波浪结构 > 动量 > 量价 > 波动率
**一致性判断：** 指标同向越多信号越强；矛盾时以趋势为主，等待动量修复。

### 5. 估值对比与合理价格区间
使用多维度估值加权（如格雷厄姆数20%、DCF各场景、PEG、行业PE等）计算**加权内在价值**，并给出"极度悲观、保守、中性、乐观、极度乐观"对应的具体合理价格区间。

### 6. 投资备忘录 (Investment Memo)
最后，必须生成一份清晰的备忘录，包括：
- **关键数据汇总**：财务、估值、技术、机构指标一览表。
- **综合投资评级**：价值投资维度(60%) + 技术分析维度(40%)，得出最终评级（A/B+/B/B-/C/D），并针对不同类型投资者（长线/中线/短线）给出明确建议。
- **主要风险点**：列出概率与影响度。
- **交易建议**：建仓区间、止损位、目标价、风险回报比。
- **催化剂与事件驱动**：近期关注事件。
- **跟踪与复盘要点**：持有期间需跟踪的关键指标及卖出信号。

## 重要约束 (Strict Constraints)
1. **所有数据必须来自实时查询**，不得使用过时数据。
2. 若某项数据查不到，需明确标注"数据缺失"，不得编造。
3. 必须标注数据的获取时间与来源。
4. 输出格式应为结构化 Markdown（可参考表格与列表排版），排版需美观。
5. **每份报告必须给出明确的投资结论，不得含糊其辞。**
6. 估值结果必须标注采用的假设前提。
7. 技术分析应参照 `docs/technical-analysis-framework.md` 中的详细规范，确保分析的系统性与专业性。
